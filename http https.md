# HTTP와 HTTPS의 차이는 무엇인가?

- HTTP의 특징
  - 암호화되지 않은 데이터를 전송하므로, 제3자가 정보를 가로챌 위험이 있음.
  - 기본 포트가 80.
- HTTPS의 특징
  - 모든 데이터를 암호화하여 전송함.
  - 기본 포트가 443.
  - 공개 키가 포함된 SSL 인증서를 사용한다.
    - 인증서 사용 시, 서버를 신뢰할 수 있게 되어 검색 엔진과 고객에게서 선호되며 성능에도 좋다.
